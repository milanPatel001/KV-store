# KV STORE
### Supports
- GET
- SET
- SET with ttl
- DEL
- Transaction - BEGIN, COMMIT and DISCARD
- Rollback for transaction
- Multiple caches (default 16)
- Sorted Sets
- Saving/Retrieving of caches on disk
- Bloom Filter

### Will Add
- LRU eviction for volatile keys on reaching threshold
- Double Ended Queue
- Geospatial Index
